To start the binary easily, an init.sh file is provided and must be executed.
The internal variables of the shell are read from ~/.rshrc.

builtin command:
- rsh (no option)
- echo (no options if you want the real echo, use /bin/echo)
- cd (no option )
- export ( use --> export x1=v1 \ x2=v2 \ x3=v3 ...)
- pwd (no option)
- history (no option)

Additional features
You can use the TAB key for command completion.
You can use the arrow keys to navigate through command history.

Syntax rules
It is forbidden to start a command with an operator.
You cannot type two operators in a row.
